Некорректное присвоение типа данных полям заполнения анкеты
Описание 

Предусловия: открыт сайт сайт http://130.193.37.179/app/pets

Тестовые данные: 
Имя: 123
Фамилия: 456
Отчество: 789
Номер телефона: 99Fgj;перч
email: котик@гугл.ком
Шаги:
1. Нажать на кнопку "без дома" в карточке "Барон, мальчик, 2 года"
2. В поле "введите имя" указать имя из Тестовых данных
3. В поле "введите фамилию" указать фамилию из Тестовых данных
4. В поле "введите отчество" указать отчество из Тестовых данных
5. В поле "+7 ваш номер" указать имя из Тестовых данных
6. В поле "your@email.com" указать email из Тестовых данных
7. В поле "комментарий" указать "хочу забрать котика"
8. Нажать на кнопку "да!"

Ожидаемый результат:
Ошибка заполнения поля "имя", допускается только текстовый формат
Ошибка заполнения поля "фамилия", допускается только текстовый формат
Ошибка заполнения поля "отчество", допускается только текстовый формат
Ошибка заполнения поля "+7 ваш номер", допускается только числовой формат
Ошибка заполнения поля "your@email.com", допускается только латиница 

Фактический результат:
Ошибка заполнения поля "+7 ваш номер", "Номер должен соответствовать +79********* !"
Ошибка заполнения поля "your@email.com", "Enter a valid email address"
